# Simulador Monte Carlo – Estimación de Probabilidad

Este proyecto implementa una simulación del método **Monte Carlo** en Java para estimar probabilidades mediante funciones y programación funcional.
El metodo de Monte Carlo es una tecnica matemática que utiliza muetreo aleatorio repetido para predecir la probabilidad de una variedad de resultados posibles en modelos que involucran incertidumbre. Se basa en repetir un modelo muchas veces, asignando valores aleatorios a las variables desconocidas en cada iteración, para generar una distribución de resultados posibles que muestren las probabilidades de cada uno.

## ¿Qué hace el programa?

Permite ejecutar **experimentos aleatorios** múltiples veces y calcular la probabilidad aproximada de que ocurra un evento.

Por ejemplo:
- Lanzar un dado y estimar la probabilidad de sacar un número mayor a 3.
- Contar cuántas veces cae cara en un volado.
- Cualquier otro evento que se pueda simular.
